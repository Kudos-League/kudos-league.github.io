# STEPS

## Download

1. Gain access to https://github.com/Kudos-League/kudos-server and https://github.com/Kudos-League/kudos-web

2. `git clone git@github.com:Kudos-League/kudos-server.git` and `git clone git@github.com:Kudos-League/kudos-league.github.io.git`

## Run backend

1. Make sure you have [Docker](https://docs.docker.com/desktop/setup/install/windows-install/) installed
2. `./scripts/start.sh local`

## Run frontend

1. Make sure you have [yarn](https://classic.yarnpkg.com/lang/en/docs/install/#mac-stable) installed
2. Run `yarn start`

## Tasks

1. Read `TASKS.md` in this folder
2. Code
3. Open Pull Request(s)

Profit!
